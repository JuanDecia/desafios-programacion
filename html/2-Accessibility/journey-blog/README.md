# Instrucciones Desafío

* Generar una estructura HTML con contenido para una página de adopción de mascotas. El archivo debe contar con la siguiente estructura:

    1. Introducir !DOCTYPE html.
    2. Generar Estructura HTML y colocar el atributo lang en "en".
    3. Completar meta información necesaria.
    4. Agregar un título a la página con el nombre "Journey Blog".
    5. En el cuerpo generar una estructura semántica correcta con los siguientes elementos obligatorios:

        * Un elemento header con un título con texto "Jota's Blog". Seguido de un elemento <h2> con texto "navigaton". Seguido un elemento <nav> y dentro del mismo un elemento <ul> con los items "My Journey", "Accessibility" y "Next Steps". Cada elemento de la lista debe estar embebido dentro de un ancla con la propiedad "href" apuntando a los subtitulos que les correspondan.

        * Un nuevo bloque con un elemento <h2> con texto "My Journey Learning to Code", seguido de un elemento <p> descriptivo. Este elemento debe apuntar al primer ancla. Seguido de un elemento <p> descriptivo. Seguido de un elemento <h3> con texto "Early Challenges" y un párrafo descriptivo. Seguido de otro elemento <h3> con texto "Breakthroughs" y un breve párrafo.

        * Un nuevo bloque que comienza con un elemento <h2> con texto "Accessibility Matters", seguido de un breve párrafo, éste elemento debe apuntar al segundo ancla de la lista. Seguido, de un elemento <h3> con texto "Screen Readers" y un breve párrafo.

        * Un nuevo bloque que comienza con un elemento <h2> con texto "What's Next?", seguido de un breve párrafo, éste elemento debe apuntar al tercer ancla de la lista. Seguido, de un elemento <h3> con texto "Coming soon: My first JavaScript project!" y un breve párrafo.

        * Finalmente un elemento <footer> que contenga un elemento <h2> con texto "Contact Me", seguido de un párrafo con texto "Email me at jotadeishon@gmail.com" el cual de sebe colocar un atributo "mailto" al mail.
