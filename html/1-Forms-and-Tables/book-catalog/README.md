# Instrucciones Desafío

* Generar una estructura HTML con contenido para una página de adopción de mascotas. El archivo debe contar con la siguiente estructura:

    1. Introducir !DOCTYPE html.
    2. Generar Estructura HTML y colocar el atributo lang en "en".
    3. Completar meta información necesaria.
    4. Agregar un título a la página con el nombre "Book Catalog Page".
    5. En el cuerpo generar una estructura semántica correcta con los siguientes elementos obligatorios:

        * Un header con un título con texto "Book Catalog".

        * Un nuevo bloque con un elemento "table" el cual contendrá 4 columnas y 6 filas.

        * Un elemento "thead", dentro de esta primera fila se deben colocar los textos "Title", "Author", "Genre" y "Publication Year". Cada uno en un elemento "th".

        * Seguido de un elemento "tbody" el cual debe contener como mínimo 4 filas las cuales deben completar la información de la tabla.

        * En la última fila se debe colocar un "tfoot" que junte todas las columnas y que tenga como información la cantidad de libros cargados.
