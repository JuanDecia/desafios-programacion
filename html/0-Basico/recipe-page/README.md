# Instrucciones Desafío

* Generar una estructura HTML con contenido para una página de adopción de mascotas. El archivo debe contar con la siguiente estructura:

    1. Introducir !DOCTYPE html.
    2. Generar Estructura HTML y colocar el atributo lang en "en".
    3. Completar meta información necesaria.
    4. Agregar un título a la página con el nombre de la receta.
    5. En el cuerpo generar una estructura semántica correcta con los siguientes elementos obligatorios:
        * Un body.
        * Un bloque principal con el elemento h1 con el nombre de la receta, seguido de un párrafo introductorio a la receta.
        * Un nuevo bloque con un elemento h2 con el texto "Ingredients", seguido de un elemento "ul" y al menos cuatro elementos "li".
        * Un nuevo bloque con un elemento h2 con el texto "Instructions", seguido de una lista ordenada con al menos cuatro elementos que enumeren los pasos de la receta.
        * Último bloque con un elemento img con una imagen válida, con su respectivo atributo "alt" y "title".
