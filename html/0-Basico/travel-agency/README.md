# Instrucciones Desafío

* Generar una estructura HTML con contenido para una página de adopción de mascotas. El archivo debe contar con la siguiente estructura:

    1. Introducir !DOCTYPE html.
    2. Generar Estructura HTML y colocar el atributo lang en "en".
    3. Completar meta información necesaria.
    4. Agregar un título a la página con el nombre "Travel Agency Page".
    5. En el cuerpo generar una estructura semántica correcta con los siguientes elementos obligatorios:

        * Un header con un título que presente el destino del viaje (Ej: Argentina). Seguido de un párrafo introductorio del país escogido.

        * Un nuevo bloque con un elemento h2 con título "Packages", seguido de un párrafo introduciendo paquetes para turistas y finalizando el bloque un elemento "ul" con dos elementos que contengan de texto "Group Travels" y "Private Tours". Ambos textos deben estar embebidos en una etiqueta ancla que no apunte a ningún lado.

        * Un nuevo bloque con un elemento h2 con el texto "Top Itineraries", seguido de al menos tres imágenes icónicas del lugar, cada una debe tener un ancla que el usuario al hacer click obtenga más información del mismo, este debe estar con un atributo "_blank", es decir se deben abrir en una pestaña nueva. Además, las imágenes deben estar en un elemento "figure" y deben tener un elemento "figcaption".
