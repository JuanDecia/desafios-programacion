# Instrucciones Desafío

* Generar una estructura HTML con contenido para una página de adopción de mascotas. El archivo debe contar con la siguiente estructura:

    1. Introducir !DOCTYPE html.
    2. Generar Estructura HTML y colocar el atributo lang en "en".
    3. Completar meta información necesaria.
    4. Agregar un título a la página con el nombre bookstore.
    5. En el cuerpo generar una estructura semántica correcta con los siguientes elementos obligatorios:
        * Un body.
        * Un bloque principal con el elemento h1 con el nombre de la bookstore, seguido de un párrafo introductorio.
        * Un nuevo bloque con un elemento h2 con un texto de alguna historia, seguido de un párrafo introductorio del libro y finalizando un boton con texto "Buy Now".
        * Otro bloque con la misma estructura pero para otro libro.
        * Último bloque con un párrafo con texto "Review your selections and continue to checkout". Seguido de dos buttons en la misma linea uno con texto "View Cart" y el otro "Checkout".
    6. Un footer con los derechos.
