# Instrucciones Desafío

* Generar una estructura HTML con contenido para una página de adopción de mascotas. El archivo debe contar con la siguiente estructura:

    1. Introducir !DOCTYPE html.
    2. Generar Estructura HTML y colocar el atributo lang en "en".
    3. Completar meta información necesaria.
    4. Agregar un título a la página con el nombre "Travel Agency Page".
    5. En el cuerpo generar una estructura semántica correcta con los siguientes elementos obligatorios:

        * Un header con un título.

        * Un nuevo bloque con dos elementos section.

        * Dentro del primer elemento section un elemento h2 que es el título del video. Seguido un elemento "video" con los atributos "controls" y "width". El atributo width debe estar seteado en "640px". Además, el vídeo debe tner un elemento "source" con un atributo "src" que apunte al video, finalizando a un atributo "type".

        * Dentro del segundo elemento section, se debe colocar un elemento "audio" con los atributos "controls", "loop" y "src".
