# Instrucciones Desafío

* Generar una estructura HTML con contenido para una página de fotos de gatos. El archivo debe contar con la siguiente estructura:

    1. Indicar la version HTML.
    2. Generar Estructura HTML.
    3. Completar meta información necesaria.
    4. Agregar un título a la página.
    5. En el cuerpo generar una estructura semántica correcta con los siguientes elementos obligatorios:
        * Un heading principal descriptivo de la página.
        * En el primer bloque de contenido, se debe colocar un H2 descriptivo. Seguido de un párrafo que contenga un link a una página que tenga fotos de gatos y una imagen de un gato.
        * Un bloque que contenga un H2 que sea una lista de gatos, seguido de un elemento H3 que refiera a cosas que le gustan a los gatos, seguido de una UL con tres elementos que contengan cosas que le gustan a los gatos y finalizando la sección con una imagen de alguna de las cosas que le gustan a los gatos. La imagen debe tener una descripción semántica.
        * El siguiente bloque se debe repetir la misma estructura que la anterior pero con cosas que los gatos odian.
        * Crear un footer con los derechos de la página.
