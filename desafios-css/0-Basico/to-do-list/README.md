**Desafío: Estilos de Enlaces en Lista de Tareas**

Estructura HTML:

✅ Debes tener una lista desordenada (<ul>) con la clase todo-list

✅ Dentro de la lista desordenada, debes tener cuatro elementos de lista (<li>). Dentro de cada elemento de lista (<li>), debe haber:

- Un elemento input con type="checkbox" y id establecido a un valor único

- Un elemento label con el atributo for establecido al id correspondiente del input

- Una lista desordenada (<ul>) con la clase sub-item

- Un elemento de lista (<li>) con un elemento de ancla (<a>) dentro

Propiedades del Enlace:

✅ El ancla debe tener:

Clase sub-item-link

Un valor href válido

Un valor target que haga que el enlace se abra en una nueva pestaña

Estilos CSS Requeridos:

✅ Los elementos de ancla no deben tener ninguna decoración de texto por defecto (sin subrayado)

✅ Debes establecer el color del texto de los enlaces a un color de tu elección

✅ Cuando tus enlaces son visitados (visited), el color debe cambiar a otro color de tu elección

✅ Cuando se pasa el cursor sobre los enlaces (hover), el color debe cambiar a otro color de tu elección

✅ Cuando los enlaces están enfocados (focus), debe haber un contorno coloreado alrededor del enlace

✅ Cuando los enlaces son clicados (active), el color debe cambiar a otro color de tu elección
