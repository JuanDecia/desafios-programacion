# 📝 Instrucciones del Desafío - Colored Boxes
Crear cinco elementos con diferentes colores usando flexbox.

## 🏗️ Estructura HTML 
Semantica Main, Header, Section, Footer:

Main:
- Contenedor principal en la escala jerárquica.

Header:
- Debe contener un elemento <h1> con título "Colored boxes".

Section:
- Debe contener los cinco elementos "box".

Footer:
- Párrafo con derechos.

## 🎨 Estilos CSS Requeridos

- Sacar los estilos por defecto del navegador con el selector universal.

- La página debe tener una fuente general y otra de respaldo.

- El elemento <main> debe estar centrado utilizando el 80%-90% como ancho máximo de la pantalla. Todo el contenido dentro de este elemento debe estar centrado tanto horizontal como verticalmente.

- Todos los elementos "box" deben tener el mismo tamaño. A su vez, cada elemento debe tener un color de fondo diferente y los bordes redondeados. Deben estar separados por al misma cantidad de espacio utilizado la propiedad "gap" del método flexbox.

- Cada elemento ".box" debe estar con un método de color diferente (hexadecimal, RGB, word-color, HSL, set-color)
