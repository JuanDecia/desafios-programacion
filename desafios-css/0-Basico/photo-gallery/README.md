# 📝 Instrucciones del Desafío - Pricing Plans
Crear una página con diferentes planes de suscripción.

## 🏗️ Estructura HTML 
Semantica <Main> -> <Header>, <Section> - <Footer>:

### Header:
- Título del desafío.

### Section:
- Un contenedor principal.
- Tres contenedores con los diferentes planes (basic, pro, premium).
- Cada card debe tener un elemento <h2> con nombre del plan, un elemento <h3> con el valor del plan. Debajo del valor una línea que lo subraye.
- Cinco características de cada plan, donde si tiene cierta característica colocar una tílde y una cruz si no la tiene.
- Un botón de seleccionar plan.

### Footer:
- Párrafo con derechos.

## 🎨 Estilos CSS Requeridos

- Formatear los estilos de la página.

- Seleccionar una fuente a gusto y una de respaldo.

- El elemento <main> debe estar centrado y todo su contenido debe estar centrado.

- Las card se deben agrandar cuando tienen el efecto de hover.

- El botón se debe cambiar a un tono más oscuro cuando tiene el efecto de hover.
