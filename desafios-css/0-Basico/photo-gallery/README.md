# 📝 Instrucciones del Desafío - Photo Gallery
Debes crear una galería de fotos utilizando flexbox.

## 🏗️ Estructura HTML 
Semantica <Main>, <Header>, <Section>, <Footer>:

### Header:
- Título con la descripción del desafío.

### Section:
- Cinco elementos <img> con fotos de gatos.

### Footer:
- Párrafo con derechos.

## 🎨 Estilos CSS Requeridos

- Formatear los estilos de la página.

- Seleccionar una fuente a gusto y una de respaldo.

- El elemento <main> debe estar centrado y todo su contenido debe estar centrado.

- Los elementos <img> deben estar con los bordes redondeados. Deben aparecer dos elementos <img> por fila.
