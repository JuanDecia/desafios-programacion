# 🟢 Nivel Básico - Fundamentos de CSS

## 🎯 Objetivos del Nivel

Dominar los estilos básicos de HTML para construir los estilos de cualquier página web.

## 🔧 Habilidades Desarrolladas

1. Selectores Básicos

- selector-elemento {} – Selección por nombre de etiqueta
- .selector-clase {} – Selección por clase
- #selector-id {} – Selección por identificador único
- * {} – Selector universal

2. Propiedades de Texto

- color – Color del texto
- font-family – Tipografía
- font-size – Tamaño de fuente
- font-weight – Grosor (negrita, normal, etc.)
- text-align – Alineación (izquierda, derecha, centro, justificado)
- text-decoration – Subrayado, tachado, etc.

3. Colores y Fondos

- background-color – Color de fondo
- background-image – Imagen de fondo
- background-repeat – Repetición de fondo
- background-size – Ajuste de la imagen de fondo

4. Box Model

- width, height – Dimensiones
- margin – Espacio exterior
- padding – Espacio interior
- border – Borde de la caja (estilo, grosor, color)

5. Listas y Enlaces

- list-style-type – Tipo de viñeta (círculo, cuadrado, número)
- list-style-image – Imagen como viñeta
- text-decoration en enlaces – Quitar o aplicar subrayado
- :hover – Estilo al pasar el cursor

6. Visualización y Posicionamiento

- display - Tipo de caja (block, inline, flex, none)
- position - Posicionamiento (static, relative, absolute, fixed)
- Ubicación Relativa/ Absoluta (top, right, bottom, left)
- z-index - Orden de apilamiento
