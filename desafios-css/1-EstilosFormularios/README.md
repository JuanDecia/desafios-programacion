# 🟢 Nivel Básico - Estilos en Formularios

## 🎯 Objetivos del Nivel

Dominar los estilos básicos en los formuularios para construir las encuestas personalizadas de cada página web.

## 🔧 Habilidades Desarrolladas

1. Selectores Básicos de Formularios

- input {} – Selector general para los campos de entrada
- input[type="tipoDeEntrada"] {} – Selector específico para un campo de entrada.
- select {} – Selector de menús desplegables
- textearea {} – Selector para las áreas de texto

2. Propiedades de Campos

- width & height – Dimensiones de los campos
- border-radius – Bordes Redondeados
- outline – Contorno al enfocar
- font-weight – Grosor (negrita, normal, etc.)
- text-align – Alineación (izquierda, derecha, centro, justificado)
- text-decoration – Subrayado, tachado, etc.

3. Estilos de Texto en Campos

- placeholder – pseudoclase(::placeholder) Selector de texto de ayuda del campo

4. Botones

- button, input[type="submit"] - Selector de etiqueta y de propiedad type
- cursor="pointer" – Estilo del cursor al pasar sobre el botón
- :hover & :active - Estilos al interactuar con el botón

5. Estados y Pseudoclases

- :focus – Estilo cuando el campo está activo
- :hover – Estilo al pasar el cursor
- :disabled – Campos deshabilitados
- :checked – Casillas o radios button seleccionados
- :valid & :invalid - Validación de campos

6. Visualización y Posicionamiento

- display - Tipo de caja (block, inline, flex, none)
- position - Posicionamiento (static, relative, absolute, fixed)
- Ubicación Relativa/ Absoluta (top, right, bottom, left)
- z-index - Orden de apilamiento
