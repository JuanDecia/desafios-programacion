# 📝 Instrucciones del Desafío - Contact Form
Simular un formulario de contacto.

## 🏗️ Estructura HTML 
Semantica: <main>, <header>, <section>, <article>, <footer>:

### Header:
- Un título con la descripción "Contact Form".

### Section:
- Un elemento <form> con los siguientes campos a requerir:
 
 * Dos entradas <input> con sus tipos específicos para: nombre y mail.

 * Un elemento <textarea>.

 * Un elemento <button> con descripción "Submit".

- Cada elemento <input> debe estar vinculado a través de un elemento <label>. Dar descripción del campo si fuera necesario.
- Los campos deben ser requeridos y deben tener una ayuda de orientación para el usuario.

### Footer:
- Párrafo con derechos.

## 🎨 Estilos CSS Requeridos

- El elemento <body> debe tener un fondo de color (preferentemente celeste).

- Una fuente general y otra de respaldo.

- El elemento <main> debe estar centrado y todo su contenido debe estar centrado. Debe tener un ancho máximo de 60%. Fondo blanco. Letras Negras.

- Los campos de los elementos <input> y <textarea> deben tener espacio y bordes redondeados. Al hacer foco el borde debe ser más grueso y de color negro. Las letras deben tener el color negro.

- El elemento <button> debe tener un fondo de color en tono verde y letras en blanco. Adicionalmente, agregarle un efecto de hover para cambiar de color el fondo a un verde más oscuro.
