# Instrucciones Desafío

* Generar una estructura HTML con contenido para una página de adopción de mascotas. El archivo debe contar con la siguiente estructura:

    1. Indicar la version HTML.
    2. Generar Estructura HTML.
    3. Completar meta información necesaria.
    4. Agregar un título a la página.
    5. En el cuerpo generar una estructura semántica correcta con los siguientes elementos obligatorios:
        * Un heading principal.
        * Tres Heading de nivel dos.
        * Luego del primer heading de nivel dos, colocar un párrafo descriptivo de la página y una imágen.
        * En el segundo y tercer heading colocar una imágen que contenga un link para ver la mascota de la sección.
