# Instrucciones Desafío

* Generar una estructura HTML con contenido para una página de adopción de mascotas. El archivo debe contar con la siguiente estructura:

    1. Introducir !DOCTYPE html.
    2. Generar Estructura HTML y colocar el atributo lang en "en".
    3. Completar meta información necesaria.
    4. Agregar un título a la página con el nombre "Final Exam Table".
    5. En el cuerpo generar una estructura semántica correcta con los siguientes elementos obligatorios:

        * Un header con un título con texto "Results Final Exam".

        * Un nuevo bloque con un elemento "table".

        * El elemento "table" debe contar con un elemento "caption" con el texto "Calculus Final Exam Grades".

        * Seguido el elemento caption se debe crear la estructura de la tabla con 7 filas y 3 columnas. El texto de la primer fila de cada columna debe estar en negritas utilizando el elemento "th". En la primera columna se coloca el texto "Last Name", en la segunda "First Name" y en la última "Grade".

        * En la séptima fila solo debe tener 2 columnas, en la primera con el texto "Average Grade" y la segunda el promedio de los 5 alumnos evaluados.

        * En el resto de filas (5) se debe completar la información de los alumnos y sus respectivas notas.
